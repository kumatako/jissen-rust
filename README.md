### 2-4 RPN計算機プログラムとデバッガによる実行
RPN（Reverse Polish Notation、逆ポーランド記法）

「XX変数をYYの値に束縛する」という表現が、まだあまり腑に落ちない。

「値(YY)に名前(XX)をつける」と言われればわかった。

「XXという名前はYYの値を表すためだけに使う」といってみると「束縛」らしくなる。
